Let's Talk About Me

Hello, I’m Lexis von Seggern a Front End Designer and Developer from Lincoln Nebraska. I create beautiful websites for any ocasion.

I love the web. You’ll find me most excited when creating beautiful websites with clear and enjoyable user experiences.

Really I enjoy each part of the process, from sitting down with a team to figure out the specs to admiring each feature of the project as it’s finally finished.

There is so much to be interested in, to learn about, and to pour one’s creativity into.
